#  Emergency Triage AI

## Overview

Emergency Triage AI is an AI-powered healthcare assistant that provides preliminary emergency risk assessment based on patient symptoms and vital signs.

The system analyzes health information and generates a structured report containing risk level, possible conditions, first-aid guidance, and hospital recommendations.

---

## Problem

During emergency situations, people may not receive immediate medical guidance due to limited healthcare accessibility and delays in initial assessment.

This project helps users understand the urgency of symptoms before reaching professional healthcare support.

---

## Features

- Patient health information collection
- AI-powered symptom analysis
- Emergency risk classification
- Possible condition prediction
- First-aid recommendations
- Healthcare guidance
- Emergency alert system
- AI-generated summary
- PDF report generation

---

## AI Implementation

The project uses a Large Language Model (LLM) through Groq API.

AI is used for:

- Natural language symptom understanding
- Risk level prediction
- Medical guidance generation
- Structured report generation
